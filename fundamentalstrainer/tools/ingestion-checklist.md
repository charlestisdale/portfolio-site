# Transcript Ingestion Checklist

For each transcript:

1. Clean captions and remove timestamps/noise.
2. Identify concepts, tools, commands, settings, scenarios, mistakes, and exam tips.
3. Search existing knowledge objects before creating a new one.
4. Merge new facts into existing concepts when appropriate.
5. Add lesson and transcript references.
6. Add relationships to adjacent concepts.
7. Mark object as `draft` until reviewed.
8. Do not generate quizzes yet.
